# Throne

## throne.stl

Throne wide-tipped pick

When searching for inspiration for a wide-tipped plectrum, which would be great for strumming and is often favoured by mandolin players, I suddenly realised there was an object in my house that had the perfect shape! *The Throne* was born!

### Dimensions

20.8 mm x 28.3 mm x 3 mm.
